# 第1章 client/server网络介绍

你打开www.google.com的时候，你的浏览器就会连接www.google.com的服务器，并且请求“/”这个页面，"/"这个页面表示网站的首页。

服务器按顺序找到这一页，如何传送给你的浏览器。

浏览器按照格式显示出来。
