过滤出需要的行。

```
sed "/SW_VERSION/p" /etc/os-release -n
```

在用awk取得对应的位置。

```
sed "/SW_VERSION/p" /etc/os-release -n | awk -F "=" '{print $2}' 
```



```
function trim() 
{ 
	echo $1 |  awk 'gsub(/^ *| *$/,"")'
}
```

```
chip=`sed -n "/vendor_id/p" /proc/cpuinfo | awk -F ":" '{print $2}'`

chip=`trim $chip`
```
