# 图形化界面

gdb可以打开图像界面的，这样调试的时候，就直观多了。

快捷键是Ctrl +X +A。在gdb界面进入退出都是这个。

这样就可以跟visual studio那样看到单步的效果了。



# 实时显示数组内容

我的数组名字是a，长度是5 。

```
display *a@5
```

效果是这样：

```
(gdb) n
1: *a@5 = {1, 2, 4, 3, 1}
1: *a@5 = {1, 2, 4, 3, 1}
```

要取消，就输入：

```
undisplay 1 #1是一个序号，上面有的。
```

如果只看一次，就用p替代display就好了。
