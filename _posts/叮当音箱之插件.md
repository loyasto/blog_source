官方默认插件

1、echo。

```
这个就是把对应的话读出来。
```

2、email。

```
这个是读取你的邮件，看看有没有未读的，有的话，给你读出来。
```

3、time。

```
这个就用来询问当前时间的。
```

4、相机。

```
可以语音调用拍照。
```



插件相当于正式音箱产品的云端部分。因为叮当是都在本地完成的。没有云端部署。
